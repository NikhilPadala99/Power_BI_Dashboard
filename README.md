

# Data Job DashBoard w/ PowerBi


![Dash board Page1](/image/Page1.png)


## Introduction

This Dashboard was created for job seekers, Job Transitioners and Job Swapper to solve a common problem: information about the data job market is scattered and hard to grasp. Using a real world dataset of 2024 of data science job postings (including titkes, salaries, and locations), this project provides a single, easy to use interface to explore market trends and compensation.

## Skills Showcased

- **Data Transformation (ETL) with power Querry :** cleaned,shaped and prepared the raw data for analysis by handling data, changing data types and creating new columns.

- **Implicit Measures :** Formulated measures to deive key insights and KPI's like `Median Yearly Salary` aand `Job Count`

- **Core Charts :** Utilized **Column,Bar,Line,** and **Area charts** to compare job count and trends overtime.

- **Geospatial Analysis :** Leveraged **Map Charts** to Visualize the global distribution of jobs.

- **KPI Indicators & Tables :** Used **Cards** to display key metrics and **Tables** for granular and sorted data.

- **Dashboard Design :** Designed an intutive and visually apealing layout, exploring both common and uncommon chart types to best tell the data story.

- **Interactive Reporting:**
     
    - **Slicers :** To dynamically filter the report by job title.
    - **Buttons :** To crate a seamless navigation experience.
    - **Drill-Through :** To navigate from a high level summary to a contextual, deatiled view.
